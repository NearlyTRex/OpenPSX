## Readme

# About
OpenPSX is an open source Sony Playstation emulator written in C/C++.

# License
OpenPSX is licensed under the GNU Lesser General Public License.

# References
Source material and documentation references and is based on PCSX, FPSE, CDRMooby,
and Pete's emulator plugins.

# Sources
The existing source files for this project are really outdated, and hopefully the
project will be continued in the future.
